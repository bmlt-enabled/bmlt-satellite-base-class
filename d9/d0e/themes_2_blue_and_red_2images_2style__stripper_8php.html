Detailed Description

This file reads in a CSS file, and optimizes it by stripping out comments and whitespace. It will also try to GZ compress the output using the standard OB functions. It can make a HUGE difference in size.

The way it works is that you call it from the <link> element (don't specify a "type" attribute), and give it a GET parameter of filename, which will equal the file path to the CSS file.

For security purposes, the file must always be a ".css" file, and you can't go out of the directory in which this file is located.
